# Vinnie's Cards

My CRUD system for cateloging and viewing my baseball card collection. Yes, I do have friends.

## Tech

- ReThinkDB
- NodeJS/Express
- React

## Setup

### Prerequisites

- You must have RethinkDB running on the system where you want to run the project
- Install packages

```sh
npm install
```

### Seed Data

```sh
npm run seed
```

### Remove Data

```sh
npm run drop
```
